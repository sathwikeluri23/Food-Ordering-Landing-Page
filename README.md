# Food Ordering Landing Page

## Overview

This project is a responsive food ordering landing page built using **HTML5** and **CSS3**. It demonstrates a modern restaurant-style interface with structured sections for navigation, featured services, menu categories, promotional content, payment options, and social media links.

The goal of this project is to showcase clean layout structuring, responsive design fundamentals, and reusable UI component patterns.

## Key Features

* Responsive navigation header
* Hero/banner section with call-to-action buttons
* “Why Choose Us” feature cards
* Explore Menu grid layout
* Two-column promotional section
* Payment methods display
* Social media follow section
* Footer with brand information
* Mobile-friendly flexbox layouts

---

## Tech Stack

* **HTML5** – semantic page structure
* **CSS3** – layout, styling, and responsiveness
* **Flexbox** – responsive alignment and grids
* **Google Fonts (Roboto)** – typography

---

## Responsiveness

The layout adapts across screen sizes using:

* Flexible containers
* Wrapping flex layouts
* Media queries for navigation alignment
* Mobile-first spacing approach
